How To Morphotools
Welcome in the Morphotools help page. You will find here all the information about how to use MorphoTools and its several functionnalities.
I. Project Management :
In order to creat your first project, you need to click "New Project". Then enter your project name (witch is mandatory) and your project description. You can now manage your projects just by clicking on them.
II. Files importation and Tool selection :
When you select a project (here My project name), you will have a new page. On this new page you will first find a button "Import" (1). Then you will have to select your images (png, jpg, ...) and your matrice (json or csv) (2). Once all your files are selected, you can send them by clicking the "Send" button (3).
With your files selected, you can choose the tool you want to use and click run.
III. Visugraph tools :
(1) Import : Load the matrix and the images after setting a edge threshold.
(2) Filter edges : Refresh the loaded matrix with a new edge threshold.
(3) Expand / Retract Edge : Default setting = retracted. When Expanded, show edges and its legend.
(4) Show / hide labels : If edges are expanded, show edges label.
(5) Refresh position : Compute and refresh the position for each nodes.
(6) Save / Load : Quick save of the actual nodes position. Or quick load of the latest saved nodes position.
(7) Export : Export JPG for exporting the jpg without cytoscape (edges and labels are not displayed). Export JPG with cy for a jpg image including edges and labels. Export JSON to download a json with the actual position for each nodes (can be load as a matrix during the importaion json/csv step).
